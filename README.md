# Work in progress hyprland ricing projecct

First attempt at using, configuring and ricing hyprland. This repository is mainly for saving progress/backups and documenting the progress of this project.

I plan on using this as my main operating system so my goals for the project are:

    - complete funtionality (settings, notifications, app switcher, everyting etc)
    - lightweight (clean and minimal with as little active processes at idle)
    - minimal aesthetic while customizable (custom wallpapers with adaptive colouring or configure your own colours)
